# Introducao
Este relatorio representa o processo de implementacao de
ferramentas na empresa [abstergo], realizado por William Gomes.
O objetivo do projeto  foi *elencar 3 serviços AWS, com a finalidade
de *realizar diminuicao de custos imediatos.

# Descrição

O projeto de implementacao de ferramentas foi dividido em
3 partes, cada uma com seus objetivos específicos. A seguir,
serão descritas as etapas do projeto:

Etapa 1:
- Amazon EC2
- Hospedar seu sistema
• Um portal onde farmácias fazem pedidos
• Área do vendedor acompanhar vendas
• Controle de estoque e distribuição


Etapa 2:
- Amazon RDS
- Banco de dados gerenciado
•  Sistemas comerciais
•  ERP farmacêutico
•  Controle de distribuição

Etapa 3
- Amazon S3
- Armazenamento de arquivos
•  Muito barato
•  Seguro
•  Escalável
•  Fácil integração com sistemas web

# Conclusao 
A implementação dessas ferramentas na empresa Abstergo Industries tem como esperado o beneficio da criação do sistema, banco de dados e armazenamento de arquivos,
que aumentará a eficiência e a produtividade da empresa.
Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por
novas tecnologias que possam melhorar ainda mais os processos da empresa.
